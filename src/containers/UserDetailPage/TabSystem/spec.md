- 解約日, o, 日付
- KID, o, 数値のみ

- システム環境, o, 選択(\*1)
- バージョン,o, 選択(\*1)
- ネットワーク, x
- モバイル利用, x
- サーバ

  - DB サーバ, o, 選択(\*2)
- ユーザーキー, o, テキスト
- DB パスワード, o, テキスト
- Fenics キー, o, テキスト
- ユーザー数(Citrix 数), o, 数値
- クライアント数(端末数), x

  - ユニバ数, o, 数値

- 端末開始 ID, o, 数値

[*2]: tableに列追加が必要

### つくるコンポーネント

- [x] テキスト入力
- [] 日付入力
- [] 数値のみ入力
- [] 数値のみ入力(追加、削除ボタンあり)
- [] セレクト入力
- [] 複数からの選択入力
